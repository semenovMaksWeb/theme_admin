# конфик получение данных на фронте
```
type: 
1 - content компонента (используется для кнопок и прочих компонентов которым в пропсе пришли данные), 
    index - используется для получение данных полученных от бэка index = номер функции
    
2 - данные компонентов

3 - данные которые генерируются на основе каких либо функции

4 - данные которые получаются через url query

id: 1 - id компонента куда нужно смотреть 
name 'id' - имя ключа объекта

result - 
    body тело запроса
    params - параметры запроса
    all - не для работы с api
```
